## 문제2. 평균 구하기
https://school.programmers.co.kr/learn/courses/30/lessons/12944
```java
class Solution {
    public double solution(int[] arr) {
        double answer = 0;
        for(int a : arr){
            answer += a;
        }
        return answer/arr.length;
    }
}
```
